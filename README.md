## What is EVON Beta?

EVON Beta is an exclusive version of the EVON Obfuscator with enhanced features for Premium users.

### Premium Features
- Level 3 (VM) Protection
- 10 MB File Limit
- 3 Second Cooldown
- Unlimited Daily Usage
- Priority Queue

## Requirements

- Windows 10/11 (64-bit)
- Discord account
- EVON Premium (gamepass purchase required)

## How it Works

1. Download and run `EVONLauncher.exe`
2. Authorize with your Discord account
3. The launcher verifies your Premium status
4. If verified, beta features are unlocked!
